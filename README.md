# flask-youtube-downloader
downloads youtube videos or extracts audio from youtube videos


set up 


```

pip3 install pipenv 
pipenv shell
```





## install dependencies 

 ``` 
 
 pip install flask
 pip install pafy
 pip install pytube
 
 ```   
 


## VS Code Formating Issues 

``` 
pipenv install pylint
pipenv install autopep8
  

```  




## running 

``` flask run ```
